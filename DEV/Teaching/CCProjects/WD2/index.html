Javascript In Your Website

Javascript is a powerful piece of the web development puzzle. Without it, many websites wouldn’t function. This isn’t the only language that adds functionality, but it is the most common in basic websites due to its simplicity and easy integration.

Adding <script> tags:

Just like learning CSS in the previous web dev course, javascript has many ways of being used in a site

  1. Create a new project folder and call it something like “javascriptwebsite”
  2. Create a subfolder called “stylesheet”.
  3. Create a file called “stylesheet.css” in that subfolder.
  4. Create the “index.html” file in the main folder (or root folder), and add the following code.
HTML Result 

<!DOCTYPE html>
<html>

  <head>
    <link rel="stylesheet" type="text/css" href="stylesheet/stylesheet.css">
    <meta charset="utf-8">
    <title>Javascripting</title>
  </head>

  <script>
    alert("Hello, World!")
  </script>

  <body>
  </body>

</html>

The <script></script> tags allow us to use javascript in our website.
Here, we added an "alert()" function that gave us a popup message when we first load in our page.

That variable’s value is determined in the <button></button> tags "onclick" property. You should see that it is equal to a function called “sayHello()” (which was defined in the <script></script> tags above.)

CSS and JS

Hover

Final thing we will cover is a form of event listener and style changing through javascript.

HTML Result 

<!DOCTYPE html>
<html>

  <head>
    <link rel="stylesheet" type="text/css" href="stylesheet/stylesheet.css">
    <meta charset="utf-8">
    <title>Javascripting</title>
  </head>

  <body>
    <h1>Change style on hover</h1>
    <p id="hoverable">Hover over me for something to happen!</p>
  </body>

  <script>
    var hoverElement = document.getElementById("hoverable");

    hoverElement.onmouseover = function(){mouseOver()};
    hoverElement.onmouseout = function(){mouseOut()};

    function mouseOver() {
      hoverElement.style.color = "red";
      hoverElement.style.textDecoration = "underline";
    }

    function mouseOut() {
      hoverElement.style.color = "black";
      hoverElement.style.textDecoration = "none";
    }
  </script>

</html>

Change style on hover

Hover over me for something to happen!

First thing to notice is that the <script></script> tag is after the body instead of before it. This is because we need the script to run after​ ​the elements have been created.

onmouseover/onmouseout: This is used to track when an element is being hovered or not

Requirements

Now that we’ve gone over some of the basic aspects of a website using Javascript, you are going to make a website based on whatever you would like! The requirements of this project are as follows:

  1. Make a basic website with HTML/CSS
    1. Follow the same rules for HTML/CSS that you learned in the last project
      1. Good file structure (index.html, sub folders, etc)
      2. Have a title
      3. Content must be in body
  2. Add one button that performs a function when clicked
    1. Button can change CSS, alter HTML content, or anything else that you think of
  3. Have at least 1 input fields with a button that does something with those values
    1. Cannot be the same calculator unless you add something new to it (like a 3rd number)
    2. Cannot be the Mad Libs game, unless you add something new to it
    3. Examples that you can do
      1. A questionnaire field asking for name, age, and other things
      2. Entering words in the field and pressing “Submit” to add those words somewhere on your page
  4. Have at least one element that does something when hovered over and when not hovered
    1. Can change CSS elements
    2. Can change words
    3. Can have an image change source
