Intro p5

p5.js is a powerful library writen in Java Script. In this class you will learn how to look at an end product, like a game, and break it down into it's base parts. By doing this we can better understand how to build common games like pong, snake, and flappy bird. And other applications like purple rain, fireworks, and simple physics.
